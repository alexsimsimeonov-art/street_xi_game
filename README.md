# Street XI Game

## Description
Street XI is an engaging and immersive soccer simulation game that allows players to experience the thrill of street soccer. Players can customize their teams, develop strategies, and compete in exciting matches against opponents.

## Setup Instructions
1. **Clone the Repository**
   ```bash
   git clone https://github.com/alexsimsimeonov-art/street_xi_game.git
   ```
2. **Navigate to the Directory**
   ```bash
   cd street_xi_game
   ```
3. **Install Dependencies**
   Make sure you have all the necessary dependencies installed. If using Node.js, run:
   ```bash
   npm install
   ```
   or for Python:
   ```bash
   pip install -r requirements.txt
   ```
4. **Run the Game**
   Depending on your setup, run:
   ```bash
   npm start
   ```
   or
   ```bash
   python main.py
   ```
5. **Enjoy the Game!**
   Start playing and have fun with Street XI!
